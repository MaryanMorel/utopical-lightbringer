How does it work?

This website displays a tweet classification made with machine learning techniques.

In few words, this is how it works:

  1. We retrieve the recents tweets of all the people your are following
  2. We explore these tweets and the metadata of the link they refers to
  3. We contruct a graph in which two users we be close and hence connected, if they use the same terms in their tweets
  4. We use a machine learning technique called spectral clustering to find clusters in this graph
  5. We display tweets by users in their respective cluster
